# AI Model Test | Realtime Overlay | YOLO

Легковесный GUI-инструмент для тестирования обученной модели компьютерного зрения **YOLO** в реальном времени прямо на экране монитора. Захватывает изображение с дисплея, выполняет инференс модели и отображает результаты поверх всех окон в виде overlay-интерфейса.

## Возможности

* **Реалтайм-инференс**: Непрерывный захват экрана и обработка кадров моделью YOLO.
* **Overlay поверх окон**: Отрисовка детекций поверх рабочего стола и активных приложений.
* **Высокая производительность**: Быстрый захват кадров через `dxcam` с высоким FPS.
* **Визуализация детекций**: Отображение bounding box'ов, названия класса и процента уверенности.
* **Цветовая индикация**: Цвет рамки изменяется в зависимости от confidence модели.
* **Быстрый выход**: Завершение работы по клавише `ESC`.

## Зачем нужен

Инструмент предназначен для быстрой проверки качества модели в реальных условиях без записи видео, подготовки датасетов и ручного анализа кадров.

Подходит для:

* тестирования модели после обучения;
* проверки precision и false positive;
* оценки скорости обработки кадров;
* визуальной отладки классов объектов.

## Требования

* Windows
* Python 3.8+
* GPU рекомендуется
* Обученная модель YOLO в формате `.pt`
* Библиотеки:

  * `ultralytics`
  * `dxcam`
  * `pywin32`
  * `numpy`
  * `Pillow`

## Установка

1. Установите необходимые зависимости:

```bash
pip install ultralytics dxcam pywin32 pillow numpy
```

2. Подготовьте модель:

   * Поместите файл весов модели (например, `best.pt`) в директорию `../` относительно скрипта или измените путь `MODEL_PATH` в коде.

## Использование

1. Запустите скрипт:

```bash
python main.py
```

2. После запуска начнётся захват экрана и отображение результатов детекции.
3. Для завершения работы нажмите `ESC`.

## Конфигурация

Основные параметры можно изменить в начале файла `main.py`:

| Параметр      | Описание                                  | Значение по умолчанию |
| :------------ | :---------------------------------------- | :-------------------- |
| `MODEL_PATH`  | Путь к файлу весов YOLO                   | `'../best.pt'`        |
| `OBJECT_NAME` | Название класса объекта для отображения   | `'proto_enemy'`       |
| `CONFIDENCE`  | Порог уверенности для фильтрации детекций | `0.5`                 |
| `FPS`         | Целевой FPS захвата экрана                | `70`                  |

## Цветовая индикация

Цвет рамки зависит от уровня уверенности модели (`conf`):

* **Красный**: Низкая уверенность.
* **Жёлтый / Оранжевый**: Средняя уверенность.
* **Зелёный**: Высокая уверенность.
